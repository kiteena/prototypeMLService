Basic Recommender Service API

In-progress APIs

Endpoints Parameters Description
/predictionspretty/ None returns json of labelled predictions of all users (currently limited to 10 results)
/predictions/ None returns json of unlabelled predictions of all users (currently limited to 10 results) /n Note: columns are user ID, item ID, true rating, and predicted rating
/predictions/<userID>/ k -- required returns k predictions for a user id
/predictionuseritem/ user id / item id -- both required returns prediction for a specific user for a given item
/rmse/ None return root mean square error of current predictions
